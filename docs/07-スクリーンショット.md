# スクリーンショット

このページでは、`/screenshots` ディレクトリに含まれる各スクリーンショットの内容と、ハードニング手順との関連を説明します。

---

## システム準備

![rhelhardeningdnfupdateandupgrade.png](../screenshots/rhelhardeningdnfupdateandupgrade.png)  
`dnf update` と `dnf upgrade` によるシステム更新。最新状態でハードニングを適用するための前準備。

![rhelhardeninggetenforce.png](../screenshots/rhelhardeninggetenforce.png)  
`getenforce` による SELinux 状態の確認。ハードニング前の Enforcing / Permissive 状態を把握。

## ユーザーとアクセス管理

![rhelhardeninguseradd.png](../screenshots/rhelhardeninguseradd.png)  
root 直接ログインを避けるため、管理用の一般ユーザーを作成した際の出力。

## SSH ハードニング

![rhelhardeningsshdconfig.png](../screenshots/rhelhardeningsshdconfig.png)  
`/etc/ssh/sshd_config` のハードニング後の設定。root ログイン無効化やパスワード認証無効化などを確認。

## ファイアウォール設定
